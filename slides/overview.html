Concept

「チャットボット」ではなく
デスクトップにキャラがいる感覚。

  • Agent の回答を会話のように読み上げ（コード・URL は読まず整形）
  • 感情に応じて表情が変化し、声に合わせて口パク
  • 透過・常駐ウィンドウで、作業しながら隣にいる
  • ユーザー像は若手・エンジニア。使っていて楽しいUI

Tech Highlight · Lip-sync

口パクを 140倍 速く。

psd-tools は可視性を変える度に全レイヤーを再合成し 約10秒。口を切り替える度に走ると実用不可だった。

10s→ 0.07s / フレーム

口を隠したベースを1回だけ合成し、口パーツを重ねるだけにした。

# 重い: 毎フレーム全再合成 (~10s) psd.composite() # 速い: ベース + 口オーバーレイ frame = base.copy() frame.alpha_composite(mouth) # → ~0.07s（キャッシュ後 0.003s）
