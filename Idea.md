# Claudeとの対話読み上げアプリ

## 機能
- ClaudeCodeでの対話読み上げ
  - Agent側の内容を読み上げる。
  - 会話しているようにするために、URLやコード等の読み上げは行わない
  - 表示するキャラごとに口調や性格を設定する（公式準拠）
  - キャラクターは現状下記の2人で、それぞれ読み上げようアプリのAPIが異なる
    - 小春六花(CeVIO AI)
    - 春日部つむぎ(VOICEVOX)
  - 読み上げようアプリはインストール済みで、パスは VoiceAppPath.md に記載
  - 各読み上げアプリの本体を立ち上げる必要がない場合（エンジン等のバックエンドだけで完結する）は、そちらを優先する。
    - このアプリともう一つ立ち上げるとふとした時に、「使ってないな～」で誤って閉じる可能性があるので。

- チャット欄機能
  - 使用するモデルの切り替え（OpusやSonnet等）
  - 対話が継続するようにセッションを継続させる
  - 「新しい話題」などのボタンでセッションを新しくする

- 立ち絵表示
 - psd画像で、キャラクターの表情等を替え、話しているように見えるようにする。
 - 読み上げに感情パラメーターを設定し、表情豊かにする。
 - キャラクターは現状下記の2名
   - [小春六花](https://tokyo6.tokyo/koharurikka/)
   - [春日部つむぎ](https://tsumugi-official.studio.site/top)

- アプリ基盤
  - Unity6000.3.10f(MCP利用せず、Editor拡張で配置等を実行できるようにする)かPython、C#ネイティブアプリ想定
  - CeVIO AI サンプルは CevioAI Sample/ に保存済み
  - [VOICEVOX](https://github.com/VOICEVOX/voicevox)

- イメージと欲しい機能
  - デスクトップ常駐で簡単に会話できる。
  - 透過ウィンドウ。
  - ウィンドウサイズ調整可能
    - 全身を表示すると大きいことがあるので、Youtube動画などのように立ち絵の上半身を移すようなこともできるイメージ
  - キャラの表示サイズ調整可能。
  - メモ機能。
  - アラーム機能。
  - ポモドーロタイマー機能。
  - デスクトップ上の位置に応じての左右反転。
  - Riot API も利用し、チームファイトタクティクスやLoLやLoLのランダムミッド等の戦績管理を行えるようにする。
  - UIも含め、サンプル的なものではなく、使っていて楽しいものにしたい。
    - ユーザーペルソナは若者やエンジニア
  - チャットウィンドウも入力欄は常時表示でもよいが、履歴は一定時間でフェードし、デスクトップにチャットボットではなく、キャラがいるように感じるようにしたい。

- その他
